The periodic table is a useful way to group elements

It has many useful properties

  • In the groups of the non transition metals, it tells you the number of electrons in the outer shell, this is useful as it can show you the charge of any ion it forms
  • Elements in groupsalso share similar chemical properties, for example group 1 reacts vigourously with water
  • Elements in a periodic table show a trend across a period

A group is a column in the periodic table
